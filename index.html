Git Command

git config --global user.email "souvik.sarkar.cs@gmail.com" --> To add email

git config --global user.name "Souvik-025" --> To add username

git config --global --list --> to List all global information

git config --list --> To list all envirenmental data

git clone --> To clone a repository in our local machine

git status --> Use check status of our git repository

git add filename --> to add new or changed file in our working directory to the git staging area.

git add . --> to add all files present in the repository

git commit -m "some message" --> to commit changes of our directory to the git repository

git push origin main --> To upload local repository content to remote repository(git repository)

init :- used to create a new git repo

git init --> Used to create a new git repo in our local repo
